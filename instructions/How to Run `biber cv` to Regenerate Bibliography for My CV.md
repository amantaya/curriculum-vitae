# 2025-06-25-08-15-05 How to Run `biber cv` to Regenerate Bibliography for My CV

My CV uses BibLaTeX to read in my list of publications that are exported from Zotero. I run the following command in a terminal to re-generate my .bib (bibliography) file which is read by LaTeX to generate a list of publications.

You may need to change your current working directory to wherever the `cv` files are located

```shell
cd latex
```

`cv` is the name of the file to run biber on to generate the bibliography

```shell
biber cv
```
